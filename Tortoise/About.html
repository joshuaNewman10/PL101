What is Tortoise?

Tortoise is an imperative programming language hooked up to the Raphael graphics library. This allows users to input commands such as forward(100); right(50); and see the resulting pictoral ouput of these commands. Essentally, the tortoises tail is a 'pen' and when he is instructed to move about a window his tail will trace his path. The Raphael graphics library is an example of 'Turtle Graphics.'

Turtle Graphics have been a useful tool in interesting young children in programming. They require the user to interact with the turtle as the language's syntax requires which eases users into a very logical and stepwise interaction. However, by giving the user immediate visual feedback of the user's commands the interaction becomes far more intuitive, immediate, enjoyable.

Behind the Scenes

Behind the scenes there are three major components at work that allow for the successful operation of this program: parsing, interpreting, and the Raphael library.

Parser

Parsing is the operation of taking a user's input sequence of commands (essentially a program) and making sure it correctly fits with the languages syntax.

Interpreter

Once the input has been syntactically verified it can be either compiled or directly interpreted. Tortoise takes the latter approach and directly interprets the parsed input. This takes the form of a recursive evaluation function.

Raphael

The final step is to actually modify the pixels on the screne by having our turtle Raphael move in response to the user's sequence of commands.
